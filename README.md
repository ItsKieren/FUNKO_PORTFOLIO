# Anime Funko Portfolio

A responsive web portfolio showcasing custom Funko Pop-style character designs from popular anime series.

## Technical Stack

- **HTML5**              : Semantic markup with accessibility considerations
- **CSS3**               : Modern features including CSS Grid, Flexbox, custom properties, and animations
- **Vanilla JavaScript** : DOM manipulation, event handling, and intersection observers
- **Responsive Design**  : Mobile-first approach with fluid typography and flexible layouts

## Collections

- **Bleach Collection**  : 26 character designs
- **Boruto Collection**  : 8 character designs

## Project Structure

```
├── index.html          # Main HTML file
├── README.md           # Project documentation
├── a.png - z.png       # Bleach character assets
├── zero.png - eight.png # Boruto character assets
└── .gitignore          # Git ignore rules
```

## Development Notes

- **CSS Architecture**   : BEM-inspired naming with component-based organization
- **JavaScript Patterns**: Event delegation and efficient DOM queries
- **Image Optimization** : Compressed PNG assets for web delivery
- **Code Quality**       : Clean, maintainable code with consistent formatting
